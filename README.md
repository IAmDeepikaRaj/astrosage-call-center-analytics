# AstroSage Call Center Analytics

Data analytics project analyzing 28,000+ call center records to guide a ₹1 crore
investment decision (hiring vs. training vs. technology).

## What's inside
- Excel workbook: pivot tables, XLOOKUP, SUMPRODUCT, array formulas, interactive dashboard
- Dashboard: 8 linked charts, 5 KPI cards, 3 slicers
- Word doc: 14 objective + 10 subjective business questions answered
- PowerPoint: full presentation walkthrough

## Key finding
58.75% of calls failed to connect — bigger than any workload or training gap —
so the recommendation prioritizes technology investment (50%) over training (30%)
and hiring/workforce optimization (20%).

## Tools used
Excel (PivotTables, CORREL, SUMPRODUCT, XLOOKUP, array formulas), data cleaning,
dashboard design
